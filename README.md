# Nova — SaaS Landing Page Template

🚀 Premium SaaS landing page for startups & developers.

## Features
- Sticky navbar with blur
- Animated hero dashboard
- Features grid, pricing, testimonials
- FAQ accordion
- Blog + integrations section
- Dark / Light mode

## How to use
1. Open index.html
2. Edit text/images
3. Deploy anywhere

## Tech
- HTML, CSS, JavaScript
- No frameworks required
